As a beginner, with some prior knowledge of ObjectOriented programming languages,
you can use the browser console as playground to try out some basic prigramming, hurray!
    On Mac + Google chrome
  1. Use cmd + option + J to open the browser console.
  2. Type in your javascript functions and hit return to view the magic on your browser.
  3. Look for all the modules that start with "Basic", to try out the most basic functions.
